Apple プラットフォーム向けのアプリをつくっています。

  • RunRun ランニングノート iOS

    月間ランニング記録アプリ

    いつものアプリで走って、RunRun で振り返ろう。Apple Watch・Strava・Nike Run Club の記録をヘルスケア経由で自動同期し、美しく可視化します。
    紹介ページ App Store

  • Seekthea iOS iPadOS macOS visionOS

    AI要約付きトレンドアグリゲーター

    あなたの興味に合いそうな記事をキーワードでスコア化し、関心の高い順に自動で並べ替え。ニュース・テック・ソーシャルのトレンドを Apple Intelligence が要約します。
    紹介ページ App Store

  • Clipnyx macOS

    軽量クリップボード履歴マネージャー

    メニューバーに常駐するクリップボード履歴マネージャー。テキスト・画像・ファイルを自動保存し、ホットキーで瞬時に呼び出せます。URL やコードは自動分類、データはすべて Mac 内に保存されます。
